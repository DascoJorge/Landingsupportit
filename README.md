# Support IT Jorsaac, SRL - Premium Landing Page

Esta es la versión premium en Modo Oscuro (Dark Luxury) diseñada para **Support IT Jorsaac, SRL**.

## Características
- **Estética Dark Luxury**: Fondos negros puros con acentos en Azul Eléctrico (#3B82F6).
- **Flip Cards Interactivas**: Tarjetas de servicios que rotan en 3D para revelar detalles técnicos.
- **Glassmorphism**: Uso extendido de efectos de "vidrio esmerilado" en tarjetas y navegación.
- **Animaciones Premium**: Efectos de flotación, scroll reveals y contadores dinámicos.
- **Estructura Integral**: Secciones para Seguridad, Redes, Soporte, Software, Energía y Obras.

## Tecnologías Usadas
- **HTML5 Semántico**
- **Vanilla CSS3** (Variables CSS, Flexbox, Grid, Animaciones 3D)
- **Vanilla JavaScript** (Intersection Observer API, DOM Manipulation)
- **Google Fonts**: Plus Jakarta Sans & Inter.
- **Imágenes**: Placeholders de alta calidad vía Unsplash (enfocados en tech y seguridad).

## Instrucciones de Personalización
1. **Logo**: Reemplaza el `.logo-placeholder` en `index.html` con tu archivo `logo.png` real.
2. **Textos**: Todos los textos son editables directamente en el `index.html`.
3. **Imágenes**: Si deseas cambiar alguna imagen, utiliza el parámetro `&w=1200` en las URLs de Unsplash para mantener la calidad.
